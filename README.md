# Portfolio Website

This is a starter portfolio website built with static HTML, CSS, and JavaScript.

## Files

- `index.html` — homepage structure
- `styles.css` — responsive design and layout
- `script.js` — mobile menu toggle and header scroll behavior

## How to use

1. Open `index.html` in your browser.
2. Replace the placeholder text and project descriptions with your own content.
3. Customize the colors, fonts, and layout in `styles.css`.

## Next steps
